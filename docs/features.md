# 機能概要

## アプリケーションの目的

ZatsuDoは、未完了のタスクが雪だるま式に蓄積する「Todo破産」を防ぎ、脳のワーキングメモリを一時的に解放するための超短期的タスク管理アプリ。
タスクを管理・整理する認知負荷を強制的なリセットによって排除し、心理的ハードルを極限まで下げて「とりあえず書き出す（ブレインダンプ）」ことに特化した環境を提供する。

### コンセプト

- とにかく雑にタスクを追加したい
- 翌朝には消えていて、重要だったらまた追加すれば良い

## 主要機能

### タスク管理（WIP）

- **自動消去（揮発性リセット）:** 設定されたタイムリミット（最大24時間、または指定したリセット時刻）を迎えると、未完了・完了を問わずすべてのタスクが一覧から自動的に消滅する。
- **タスク状態のトグル:** タスクの完了（Done）/未完了のシンプルな切り替え。
- **一覧表示（Dump / Today）:** 今、目の前にあるタスクのみをノイズレスに表示する。

### タスク追加（WIP）

- **クイック入力:** アプリを開いてすぐに書き込める、最短ステップの入力UI。
- **入力項目の最小化:** テキストのみ。カテゴリ、タグ、優先度、複雑な期限設定などは意図的に排除し、「雑に書く」ことを促進する。

### 設定

- **リセット時刻の設定:** タスク一覧のツールバーから設定シートを開き、リセット時刻を変更できる（デフォルト: 午前4時）。設定値は `AppSettingsStore` 経由で UserDefaults に永続化される。
- **リセットルールのカスタマイズ（WIP）:** ユーザーの生活スタイルに合わせて、タスクが消滅するタイミングを設定する。
  - 例：毎日特定の時刻にリセット（午前4時など）
  - 例：タスク作成から指定時間後にリセット（最大24時間）

### オンボーディング

- 初回起動時のみ、アプリのコンセプトを紹介する2ページのスライドを表示する。
  - ページ1: 「雑に書き出す」 — 脳内のノイズを吐き出す場所
  - ページ2: 「翌朝には消える」 — タスクが蓄積しない安心感
- 完了後はタスク一覧画面に遷移し、以降はオンボーディングをスキップする。
- 完了状態は `AppSettingsStore` 経由で UserDefaults に永続化される。
